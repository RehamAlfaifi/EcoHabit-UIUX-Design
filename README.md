# Eco-Recycling-App

A mobile application prototype designed to encourage recycling and eco-friendly habits through interactive and motivational features.

## Features
- User-Friendly Interface
- Interactive Prototype
- Recycling Challenges
- Reward and Motivation System
- Progress Tracking
- Eco Awareness Content
